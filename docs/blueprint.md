# **App Name**: CohortFlow

## Core Features:

- Public Application Portal: A dedicated public-facing page for prospective students to apply, capturing necessary demographic and motivational data.
- Admin Application Management: An administrator interface to review, approve, or reject student applications, facilitating movement to the student database upon acceptance and payment.
- Cohort Creation & Management: Admin tools for creating, configuring, and managing distinct, time-bound educational cohorts with specific themes and timelines.
- Personalized Student Calendar: A student-centric dashboard displaying synchronous session links (e.g., Zoom, Google Meet) and assignment deadlines, automatically adjusting to the student's local timezone.
- AI-Enhanced Peer Review Coordination: A system that coordinates random pairing of students within cohorts for peer review, and provides a generative AI tool for summarizing and extracting key insights from submitted peer feedback.
- Gated Alumni Directory: A secure, cross-cohort directory for past students to network, accessible only after successful completion of their bootcamp.

## Style Guidelines:

- Primary color: A deep and mature blue (#2966A3), chosen to convey professionalism, trust, and academic rigor.
- Background color: A subtle, light blue-grey (#F0F2F5) that provides a clean canvas and visual calm, derived from the primary hue.
- Accent color: A vibrant, analogous medium purple (#6767E4), used sparingly to draw attention to key actions and interactive elements, ensuring strong visual contrast.
- Headline and Body font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics across various content types, suitable for both extensive text and bold headings.
- Employ a consistent set of minimalist line icons or clean flat-filled icons to maintain a professional and intuitive user experience across all modules.
- Implement a structured and spacious layout with ample whitespace, emphasizing clarity and organization essential for high-ticket educational content.
- Utilize subtle and purposeful animations for transitions and feedback, enhancing user interaction without distracting from the core content or professional tone.
