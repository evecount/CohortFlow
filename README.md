# CohortFlow | Operating System for Elite Academies

CohortFlow is a professional, white-label infrastructure designed for experts and academy founders to launch high-ticket, cohort-based educational programs. It provides the "pickaxes" to help you scale your knowledge without the operational headache.

## 🧬 The Vision
We are building more than an LMS; CohortFlow is a **massive deep learning pipeline for multimodal research**, leveraging AI to bridge the gap between academic theory and professional practice.

## 🚀 Key Features

### 👑 For Academy Founders (Admin)
- **AI Course Builder**: Transform research PDFs into structured curricula with cited slide outlines.
- **AI Rehearsal Lab**: A multimodal coach using Gemini Live to perfect presentation delivery through video and audio analysis.
- **Production Suite**: Generate AI voiceovers (TTS) and cinematic video teasers (Veo) in seconds.
- **AI Admissions Funnel**: Automatically score and vet applicants for "fit" and "signal" using custom criteria.

### 🎓 For Students (Dashboard)
- **Personalized Learning**: Real-time schedules and localized workshop times.
- **AI Peer Review Engine**: AI-summarized insights from cohort peers to digest feedback loops quickly.
- **Alumni Networking**: Exclusive access to a directory of high-performers.

## 🛠 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **AI Engine**: Genkit with Google Gemini 2.5 & Veo
- **Styling**: Tailwind CSS & ShadCN UI
- **Database/Auth**: Firebase (Production Ready)
